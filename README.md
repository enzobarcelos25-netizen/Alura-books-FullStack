# Alura Books FullStack

Projeto de estudo full stack com front-end React e back-end Node.js para uma aplicacao de livros e favoritos.

## Objetivo

Praticar a integracao entre interface React e uma API Node/Express, separando responsabilidades entre front-end, rotas, controladores e servicos.

## Funcionalidades

- Interface de livros com React.
- Pesquisa e exibicao de livros.
- Pagina de favoritos.
- API para listar, buscar e gerenciar livros/favoritos.
- Dados persistidos em arquivos JSON no back-end.

## Stack

- React
- JavaScript
- Node.js
- Express
- JSON como base de dados simples

## Estrutura

```txt
Alura-books-FullStack/
  backend/
    alura-books-server/
      app.js
      rotas/
      controladores/
      servicos/
      livros.json
      favoritos.json
  frontend/
    src/
      componentes/
      rotas/
      servicos/
```

## Como rodar

### Back-end

```bash
cd backend/alura-books-server
npm install
npm start
```

### Front-end

```bash
cd frontend
npm install
npm start
```

## Aprendizados praticados

- Criacao de API com Node.js.
- Organizacao em rotas, controladores e servicos.
- Consumo de API no React.
- Separacao de front-end e back-end.
- Manipulacao de arquivos JSON como fonte de dados.
